Lancy Leilões

Projeto

O aplicativo Lancy Leilões é uma plataforma voltada tanto para a realização quanto para a participação de leilões de maneira remota.
Inicialmente o projeto foi pensado para o nicho de galerias de artes. Entretanto, pode abranger todo o universo de leilões de pequenas, grandes empresas e até pessoas físicas.

Responsabilidades

Pesquisa de UX, Arquitetura de informação, Escrita de UX, Prototipagem, Design Visual UI/UX e Análise de Usabilidade.
